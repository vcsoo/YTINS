# YTINS 홈페이지

㈜와이티아이엔에스(YTINS) 회사소개 홈페이지 저장소입니다.

- Application 개발 SI · IT 아웃소싱 · 비즈니스 솔루션 · 스마트 서비스(BigData / AI / IoT / Cloud) 전문기업
- 정적 HTML 사이트로 구성되며, Cloudflare Pages 를 통해 배포됩니다.

## 배포

이 저장소는 Cloudflare Pages 에 GitHub 연동으로 연결되어 있으며,
지정된 프로덕션 브랜치에 푸시하면 자동으로 배포됩니다.

## 구조

- `index.html` — 메인(회사소개) 페이지
- 정적 자산(이미지, CSS, JS)은 루트 또는 하위 폴더에 배치
